# Pathway InterIIT 14.0 - Agentic AI

## Setup Instructions
1. Clone the repository
```bash
git clone https://github.com/Quantum-Codes/Pathway-InterIIT-14.git
cd Pathway-InterIIT-14
```
## Getting Started

Follow these steps in order to start the application:

### Step 1: Backend

1. Navigate to the `backend` directory via the `cd` command
2. Follow the instructions in the Backend's local README
3. Ensure the backend service is running and accessible before proceeding

### Step 2: Pipeline & Database

1. Navigate to the `pipeline` directory via the `cd` command
2. Follow the instructions in its local README to set up the data pipeline and database
3. Confirm all data services (Kafka, PostgreSQL) are stable and running

### Step 3: Frontend

1. Navigate to the `frontend` directory via the `cd` command
2. Follow the instructions in the Frontend's local README to start the user interface
3. Open your browser and access the application
